**Я заметил:** форма в `index.php` отправляет запрос на `events.php`, а этот скрипт возвращает JSON. Поэтому вместо читаемой страницы я вижу сырой JSON.

**Моё решение:**

- В файле `index.php` меняю `action="events.php"` на `action="eventsget.php"`.
- При необходимости добавляю `target="_blank"`, чтобы результаты открывались в новом окне.

Остальное не трогаю — `events.php` оставляю как API.

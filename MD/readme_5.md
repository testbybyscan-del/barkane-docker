```json
{
  "name": "Концерт в Ялте",
  "venue": "концертный зал",
  "theme": "музыка",
  "event_type": "концерт",
  "city": "Ялта",
  "date": "2026-07-15",
  "for_kids": false,
  "is_free": false,
  "has_food": true,
  "has_alcohol": true,
  "has_transfer": false
}
```

**Поля с описанием:**


| Поле       | Тип  | Обязательное               | Описание                                                                                                                                                                                                       |
| -------------- | ------- | -------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `name`         | string  | Да                                   | Название мероприятия                                                                                                                                                                                |
| `venue`        | string  | Да                                   | Место проведения (концертный зал, ночной клуб, вне помещений, на берегу моря, на яхте, музей)                                                 |
| `theme`        | string  | Да                                   | Тематика (музыка, искусство, кулинария, здоровье, спорт, авто/мото, альпинизм, история, этнография, общество, экология) |
| `event_type`   | string  | Да                                   | Тип (концерт, лекция, выставка, мастеркласс, яхтинг, творческий вечер, дискотека, состязание)                                               |
| `city`         | string  | Да                                   | Топоним Крыма (из списка в форме)                                                                                                                                                            |
| `date`         | string  | Да                                   | Дата в формате`YYYY-MM-DD`                                                                                                                                                                                 |
| `for_kids`     | boolean | Нет (по умолчанию false) | Для детей                                                                                                                                                                                                      |
| `is_free`      | boolean | Нет (по умолчанию false) | Бесплатное                                                                                                                                                                                                   |
| `has_food`     | boolean | Нет (по умолчанию false) | С фуршетом                                                                                                                                                                                                    |
| `has_alcohol`  | boolean | Нет (по умолчанию false) | С алкоголем                                                                                                                                                                                                  |
| `has_transfer` | boolean | Нет (по умолчанию false) | С трансфером                                                                                                                                                                                                |

---

**Примечание:**

- Все строковые поля должны быть переданы в кавычках.
- Булевы значения передаются как `true`/`false` (без кавычек).
- Дата строго в формате `ГГГГ-ММ-ДД`.
- Все поля обязательны, кроме булевых – они имеют значение по умолчанию `false`, если не указаны.
